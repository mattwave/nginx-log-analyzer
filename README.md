# nginx-log-analyzer
Analyzin' Logs

How to use it

    Create the file:
    nano nginx-log-analyzer.sh

    Paste the code above into the editor and save.

    Make it executable:
    chmod +x nginx-log-analyzer.sh

    Run it against your logs:
    ./nginx-log-analyzer.sh /var/log/nginx/access.log
